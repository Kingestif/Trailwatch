## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended)

### Installation

```sh
pnpm install
```

### Environment Variables

Create a `.env` file in the root directory:

```
TMDB_API_KEY=your_tmdb_api_key
```

### Development

```sh
pnpm dev
```

### Build

```sh
pnpm build
```

### Start

```sh
pnpm start
```

## Folder Structure

```
app/
	api/           # API routes
	home/          # Home page components
	ui/            # UI components
	lib/           # Shared types and placeholder data
public/          # Static assets
```
